what is css?

1. CSS is the language we use to style an HTML document.

2. CSS describes how HTML elements should be displayed.

3. CSS stands for Cascading Style Sheets.

4. CSS describes how HTML elements are to be displayed on screen, paper, or in other media

5. CSS saves a lot of work. It can control the layout of multiple web pages all at once

Rules of using css framework.

There are three method of using css in your work space.

1. inline method

2. internal method

3. external method

Inline Method => This method specify how you can style your element with CSS using the attribute format. An inline CSS is used to apply a unique style to a single HTML element. An inline CSS uses the style attribute of an HTML element Don't know what attribute is? Check it out now.

Internal Method => An internal CSS is used to define a style for a single HTML page. An internal CSS is defined in the head section of an HTML page, within a style element.

External Method => An external style sheet is used to define the style for many HTML pages. To use an external style sheet, add a link to it in the head section of each HTML page:
